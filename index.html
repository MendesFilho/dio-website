O que é a Web?

(Voltar)

Imagine uma teia de aranha gigante... Agora pense em milhões de computadores interligados. Isso é a web!

A web, ou World Wide Web, é um sistema de documentos interconectados acessíveis via internet.

Quem foi o pai da web?

(Voltar)

O pai da web é o Tim Berners-Lee, um verdadeiro gênio britânico! Lá em 1989, ele teve uma ideia brilhante: combinar hipertexto com a internet, criando a World Wide Web.

Imagina só, ele basicamente transformou uma rede de computadores

em uma teia gigante onde todo mundo pode compartilhar informações, fotos de gatinhos e memes! Hoje, graças ao Tim,

podemos navegar na web, fazer compras online e nos perder em vídeos engraçados no YouTube. Então, quando você estiver

navegando, agradeça ao Tim Berners-Lee por essa invenção fantástica!

URLs e Navegadores

(Voltar)

URLs são tipo os endereços das casas na web. Sabe aquele "http://"? Ele te leva direto para o site que você quer

visitar, como se fosse uma placa de rua digital. Já os navegadores são as janelas que você abre para espiar o que

tem nessas casas. Tem vários estilos: o Chrome é rapidão, o Firefox é o amigo do meio ambiente, e o Safari é chique

e exclusivo. Eles transformam aqueles códigos complicados em páginas bonitinhas e interativas. Sem eles, a web

seria um lugar bem sem graça, só com textos e números. Então, agradeça ao seu navegador favorito por te levar numa

viagem divertida pela internet todos os dias!

HTML, CSS e JavaScript

(Voltar)

HTML, CSS e JavaScript são tipo os super-heróis dos sites! O HTML é o construtor, ele cria a estrutura, como os tijolos

de uma casa. O CSS é o estilista, ele deixa tudo bonito e colorido, escolhendo as roupas e a decoração. Já o JavaScript é

o mágico da turma, ele faz tudo ganhar vida, com animações e interações incríveis. Juntos, eles transformam linhas de código

em páginas de web que a gente adora navegar. Sem o HTML, os sites seriam só um monte de palavras jogadas. Sem o CSS, seriam

super sem graça, só preto e branco. E sem o JavaScript, nada se mexeria, seria tudo parado. Esses três são a trinca dinâmica da internet!

Para parágrafos usamos a tag p

Os navegadores adicionam automaticamente uma única linha em branco antes e depois de cada "tag p" elemento.

Para citações usamos a tag blockquote

A "tag blockquote" especifica uma seção citada de outra fonte.

HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Define o significado e a

estrutura do conteúdo da web. Outras tecnologias além do HTML geralmente são usadas para descrever a aparência/apresentação

(CSS) ou a funcionalidade/comportamento (JavaScript) de uma página da web.

Para deletar e inserir um texto usamos a tag del

A tag define o texto que foi excluído de um documento. Os navegadores geralmente traçam uma linha no texto excluído.
